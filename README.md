# micropython-mcp3462

A MicroPython driver for the [MCP3462 ADC](http://ww1.microchip.com/downloads/en/DeviceDoc/MCP3461-2-4-Two-Four-Eight-Channel-153.6%20ksps-Low-Noise-16-Bit-Delta-Sigma-ADC-DS20006180B.pdf)

The MCP3462 converters are 16-bit Delta-Sigma Analog-to-Digital Converters
